Class ToggleableContainer<T>

An array providing an API to easily toggle-insert an element - meaning, add the item if it isn't present, or remove it if it is.

toggle

  • toggle(element: T): void
    • Defined in src/data/ToggleableContainer.ts:68

    Either adds or removes an element from this container - whatever is opposite of the current state.

    example

    const result = new ToggleableContainer(); result.toggle(4); console.log(result.toArray()); // [4] result.toggle(4); console.log(result.toArray()); // []
