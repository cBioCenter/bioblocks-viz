chell-viz

Chell Visualization Library

Intro

Visualization components for use in Human Cell Atlas portals, such as with Chell.

Check out the API Docs or some live examples!

Getting Started (For Development Use)

Please see the Getting Started page for an introduction to the chell-viz development environment!

Getting Started (For Non-Development Use)

  • Clone the repository:
git clone https://github.com/cBioCenter/chell-viz
  • Install dependencies with yarn:
cd chell-viz
yarn
  • Start a local server to see the components in action!
yarn start
